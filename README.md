# Simple-Bank-System
Project Description
This is a beginner-friendly Java console application that simulates a basic banking system. It allows users to manage a digital wallet using simple command-line inputs. This project was developed to practice fundamental programming concepts while studying Computer Science at Algoma University.

Core Features
Balance Tracking: Keeps a real-time record of your total funds.
Add Funds: Simple addition logic to increase your balance.
Subtract Funds: Basic subtraction logic for withdrawals.
User-Friendly Interface: Uses simple text commands like "add" and "sub" to interact with the system.

Technical Skills Used
Java Programming Language: Developed entirely in Java.
Input Handling: Utilizes the Scanner class to read user commands.
Control Flow: Uses while loops and if-else statements for logical decision-making.
Arithmetic Operations: Demonstrates basic mathematical logic in a software environment.
